# 提案された変更を含むテキスト

テキストは{--削除--}および置換テキスト{++追加++}にすることができます。 これも可能です
{~~削除~>追加~~}操作に結合されます。 {==ハイライト==}も
可能な{>>そしてコメントはインラインで追加することができます<<}。

{==

開閉を入れることでブロックにもフォーマットを適用できます
別々の行にタグを付け、タグとコンテンツの間に新しい行を追加します。

==}





# 強調表示されたテキスト

- ==この文章は強調されます==
- ^^この文章は挿入されました^^
- ~~この文章は削除されました~~



# 下付き文字と上付き文字を含むテキスト

- H~2~0
- A^T^A
